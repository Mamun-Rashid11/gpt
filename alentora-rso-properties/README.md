# Alentora RSO Properties

WordPress plugin to integrate with the Resales Online API and display property listings using a shortcode.

## Shortcode

`[alentora_properties]` - Displays properties from the default sales filter. Optional attributes:

- `filter` – Override filter ID.
- `per_page` – Number of properties per page.
- `page` – Page number (used internally for pagination).

## Settings

Configure API credentials and filter IDs under **Settings > Alentora RSO** in the WordPress admin.
